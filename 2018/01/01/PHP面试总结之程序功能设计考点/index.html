PHP面试总结之程序功能设计考点

发表于 2018-01-01 | 分类于 面试

1、回顾

真题：编写一个在线留言本，实现用户的在线留言功能，留言信息存储到数据库，要求设计数据表内容以及使用PHP编码完成。
考官考点：数据表设计，数据表创建语句，选择PHP连接数据库的方式，编码能力

2、数据表设计

分析数据表结构：留言信息：ID，留言标题，留言内容，留言时间，留言人

3、数据表的创建

留言本表message：create table message( … ) ENGINE=InnoDB DEFAULT CHARSET=utf8;

4、选择PHP连接数据库的方式

PDO：可扩展性更好、支持预处理、面向对象
MySQLi：只支持MySQL操作、支持预处理、面向对象和过程，效率较高
mysql：只支持MySQL数据库、没有预处理的支持、面向过程

5、编码能力

PDO的基本操作

6、真题

设计一个无限分类表(提示：1.id title pid ，然后使用递归或加个字段path)
